# 大转盘 — 设计规范文档

> 版本：v1.0  
> 更新日期：2024-06-15

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **极简** | 一个屏幕只做一件事，减少选择负担 |
| **直觉** | 不需要说明文字，一看就懂怎么用 |
| **温暖** | 淡棕色调带来舒适、食物相关的亲切感 |
| **大触控** | 所有可点击元素至少 44×44px |

## 2. 配色方案

### 2.1 主色调

| 用途 | 色值 | 色块 | 说明 |
|------|------|------|------|
| 页面背景 | `#faf7f2` | ████ | 暖白/极淡棕 |
| 卡片背景 | `#f5f0e8` | ████ | 淡棕 |
| 卡片边框 | `#e0d5c5` | ████ | 浅棕边框 |
| 主文字 | `#5d4e37` | ████ | 深棕 |
| 次要文字 | `#8b7355` | ████ | 中棕 |
| 主按钮 | `#8b7355` | ████ | 中棕底 + 白字 |
| 按钮悬停 | `#6d5a42` | ████ | 深棕 |
| 危险按钮 | `#c0392b` | ████ | 暗红（删除用） |
| 确认按钮 | `#5a8f5a` | ████ | 暗绿（保存用） |

### 2.2 转盘扇区色盘

```

#e6c9b0  #d9b898  #c9a882  #b8956e
#f0dcc0  #e8d0a8
```

共 10 种暖色调，按选项数量循环使用，相邻扇区不重复同色。

### 2.3 功能色

## 3. 排版

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 24px | 700 | 页面标题 |
| H2 | 18px | 600 | 区块标题 |
| Body | 16px | 400 | 正文、选项文字 |
| Small | 13px | 400 | 辅助信息、日期 |
| Caption | 11px | 400 | 图片标注 |

字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`

| 层级 | 值 | 用途 |

## 7. 布局结构（移动优先）

```
┌──────────────────────────────┐
│  🎯 今天吃/喝什么？          │  ← header
│  [🍜 吃的] [🧋 喝的]        │  ← 分类标签
├──────────────────────────────┤
│                              │
│        ▼ （指针）            │
│     ┌──────────────┐        │
│     │              │        │
│     │   转盘区域    │        │  ← Canvas
│     │   (圆形)     │        │
│     │              │        │
│     └──────────────┘        │
│                              │
│    [ 🎯 开始 ]              │  ← 旋转按钮
├──────────────────────────────┤
│  📝 选项  |  📋 记录        │  ← 底部导航
└──────────────────────────────┘
```

- 最大宽度：480px
- 水平居中
- 转盘 Canvas 占屏幕宽度的 80%（移动端约 300px）

## 8. 交互规范

| 交互 | 反馈 |
|------|------|
| 按钮点击 | 轻微缩放（scale 0.95）+ 颜色变深 |
| 转盘旋转 | 3-3.5 秒缓出动画 |
| 旋转中 | 按钮禁用，灰色显示 |
| 结果弹出 | 淡入弹窗，1.5 秒后或点击关闭 |
| 标签切换 | 下划线滑动 + 颜色过渡 |
| 删除选项 | 左滑露出删除按钮（移动端）/ 悬停显示 ×（桌面端） |
